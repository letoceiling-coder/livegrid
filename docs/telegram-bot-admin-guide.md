# LiveGrid Telegram Bot — руководство для администратора

Документ описывает, как настроить Telegram-бота через CRM без правки `.env`.

## Где находятся настройки

1. Откройте CRM: `/crm/settings`.
2. Перейдите в блок **Telegram Bot интеграция**.
3. Заполните поля:
   - **URL endpoint бота** — полный URL до `POST /internal/notify`.
   - **Внутренний токен** — опционально, если в боте включена проверка `x-internal-token`.
4. Нажмите **Сохранить**.
5. Нажмите **Проверить интеграцию** — в канал должно прийти тестовое уведомление.

## Что это включает

- Уведомления о новых регистрациях пользователей.
- Уведомления о новых заявках (после подключения конкретных форм).

## Проверка работоспособности

1. Убедитесь, что бот запущен и доступен по `notifyUrl`.
2. В CRM сохраните URL/токен.
3. Зарегистрируйте нового пользователя на сайте.
4. Проверьте, что в Telegram-канал пришло уведомление `👤 Новый пользователь!`.
5. Для быстрой проверки без регистрации используйте кнопку **Проверить интеграцию** в CRM.

## Требования к безопасности

- Используйте HTTPS URL.
- Для production рекомендуется всегда задавать токен.
- Не публикуйте токен в открытых каналах и тикетах.

## Что делать при ошибках

- Если уведомления не приходят:
  - проверьте корректность `notifyUrl`;
  - проверьте токен (должен совпадать с ботом);
  - проверьте доступность бота извне (firewall/reverse proxy);
  - проверьте логи backend и telegram-bot.
